# PhotoHope Website Project Notes

### Sections
- **Navigation Bar (`<nav>`)**: Holds the navigation elements.
- **Header (`<header>`)**: Hero section displaying headings and a call-to-action button.
- **Services Section (`<section class="services">`)**: Displays various photography services using cards.
- **Why Us Section (`<section class="why-us">`)**: Lists reasons to choose PhotoHope, with an image of the owner.
- **Project Details Section (`<section class="project-details">`)**: Provides detailed information about different photography projects.

### Form
- **Form (`<form>`)**: Allows users to input their details for submission.

### Footer
- **Footer (`<footer>`)**: Contains links to additional pages.

## CSS Explanation

### General Styles
- **Reset Styles**: Resets default margin, padding, and box-sizing.
- **Root Variables**: Defines custom variables for color and font.
- **Body Background**: Sets the background color of the webpage.

### Navigation Styles
- **Navigation Bar Styles**: Styles navigation elements for alignment and appearance.
- **List Item Styles**: Styles navigation links and defines hover effects.

### Header Styles
- **Hero Section Styles**: Styles hero section with a background image, headings, and a call-to-action button.

### Services Styles
- **Card Styles**: Defines styles for service cards, including images and descriptions.

### Why Us Section Styles
- **List Styles**: Styles reasons to choose PhotoHope with numbered lists and hover effects.
- **Owner Image Styles**: Styles owner image with a border and hover effect.

### Project Details Styles
- **Container Styles**: Styles layout of project details, including images and descriptions.

### Form Styles
- **Form Styles**: Styles form elements for better user experience.

### Footer Styles
- **Footer Styles**: Styles footer for alignment and appearance.
